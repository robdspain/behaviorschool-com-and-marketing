Observations Pro is a local-first iPhone app for recording structured behavior observations.

Getting Started

  1. Open the app and complete local workspace onboarding.
  2. Add students from the Students tab.
  3. Add behavior targets from a student profile.
  4. Start an interval, ABC, or frequency observation from the Record tab.
  5. End the observation to review and share the session summary.

Local Data

Version 1.0 stores workspace data on the device. There is no account login, school sync, or web dashboard connection in the first App Store release.

Troubleshooting

  • If an active observation was interrupted, return to the Record tab and use Resume Saved Session.
  • If onboarding needs to be viewed again, open Settings and choose Restart Onboarding. This resets first-run screens without deleting local app data.
  • If haptic feedback is not desired, turn it off in Settings.
