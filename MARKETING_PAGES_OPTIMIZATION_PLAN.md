## Conversion Optimization Elements

### Above-the-Fold Must-Haves:
1. Clear H1 stating exact practice type
2. Primary CTA button (large, high-contrast)
3. "Free, no signup" badge
4. Quick benefit bullets (3-4 items)

### Social Proof:
- Pass rate statistics
- User testimonials
- "X candidates practiced this week"
- Trust badges (BACB 6th Edition)

### Urgency/Scarcity:
- "Start practicing today"
- "Join X others studying now"
- "Limited time: All domains free"

### Risk Reversal:
- "100% free forever"
- "No credit card required"
- "Cancel anytime" (for paid upgrades)
- "Privacy guaranteed"
